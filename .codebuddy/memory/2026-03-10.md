## 修复考试测验 bug（用户无法答题）

修复了 `src/utils/quizGenerator.js` 和 `src/pages/QuizPage.jsx` 中的多个 bug：

1. **`generateWordMatching` 原地修改 scenes 数组**（根本原因）：`scenes.sort()` 会变异传入的数组，导致后续题目生成器使用被打乱的数据，可能产生异常。修复为 `[...scenes].sort()`
2. **`generateGrammarChoice` 可能产生不足选项**：当字符串替换无变化时，`wrongOptions` 可能为空。增加了更多变换策略和回退机制
3. **`generateQuiz` 缺少错误处理**：单题生成抛异常会导致整个测验生成失败，quiz 为 null 但 UI 无提示。添加了 try/catch 和替补逻辑
4. **`startQuiz` 无空检查**：`generateQuiz` 返回 null 时直接设置到 state 会导致后续渲染崩溃。添加了空检查和用户提示
5. **词汇配对题交互改善**：已选含义显示禁用状态，支持点击已配对单词取消选择
6. **各生成器添加 keywords 防御性检查**

构建和 lint 验证通过。
