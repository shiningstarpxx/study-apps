## Billions English

一个基于《亿万》台词的英语学习网页应用，核心目标是把 **真实影视语境**、**每日进度管理**、**AI 驱动的苏格拉底式提问**、**间隔重复复习** 和 **测验闭环** 组合成一套完整学习系统。

### 项目定位

- **目标用户**：想提升美剧听说读写、商务表达、金融/法律语感的中高级英语学习者
- **核心素材**：以《亿万》风格台词为学习切片，按“角色 - 场景 - 句子 - 词汇 - 语法 - 思辨”组织
- **学习原则**：不只背单词，而是通过语境理解、主动回忆、复盘反馈形成长期记忆

### 当前已实现能力

- **台词学习**：逐句学习、中文释义切换、关键词高亮、语法点展示。已完整载入《亿万》第一季课程。
- **每日任务**：学习新台词、复习旧词汇、完成测验、进行苏格拉底对话
- **SRS 复习**：基于简化 SM-2 算法安排词汇复习
- **考试系统**：词汇选择、填空、翻译、语法、场景理解等题型
- **苏格拉底学习 (AI 版)**：接入真实的 Gemini 1.5 大模型，通过动态的引导式问题做理解深化、词汇迁移和语法发现，拒绝死板预设。
- **学习统计**：等级、XP、连续学习、测验记录、剧集进度
- **本地持久化**：学习进度和 AI API Key 安全保存在浏览器 `localStorage`
- **真实路由**：支持基于 URL 的页面切换，适合演示和静态部署
- **跨平台支持**：已通过 Capacitor 集成 iOS 平台支持

### 页面结构

- **`/`**：学习首页 Dashboard
- **`/learn`**：台词逐句学习，支持集数导航与进度追踪
- **`/review`**：间隔重复复习
- **`/quiz`**：考试与测验
- **`/socratic`**：基于 Gemini 的真实 AI 苏格拉底式引导学习
- **`/vocabulary`**：词汇本
- **`/stats`**：统计与学习方法论

### 这套方案借鉴了哪些顶级学习方法

#### 1. 可理解输入（Comprehensible Input）
- 学习材料来自真实场景，而不是脱离语境的孤立句子
- 每句台词同时附带角色、场景、语义和翻译，降低理解门槛
- 难度分层让学习者处于接近 \\(i+1\\) 的最佳学习区间

#### 2. 间隔重复（Spaced Repetition）
- 新学词汇会进入 SRS 复习系统
- 学习者根据记忆强度打分，系统自动调整下一次出现时间
- 目标是减少低效重复，强化长期记忆

#### 3. 主动回忆（Active Recall）
- 通过测验、填空、翻译和词汇回忆代替被动浏览
- 学习不是“看懂”，而是“能主动提取”
- 测试本身就是学习的一部分

#### 4. 苏格拉底式教学（Socratic Method）
- 不直接给答案，而是通过连续提问引导思考
- 适合训练：语义推理、语用理解、文化比较、表达迁移
- 能把“知道意思”升级成“能自己分析语言”

#### 5. 情境学习（Situated Learning）
- 以角色目标、冲突关系、职场语境为背景理解表达
- 让台词不只是字幕，而是完整的交流行为
- 对影视英语、商务表达、谈判措辞特别有效

#### 6. 检索练习 + 反馈闭环
- 学 -> 练 -> 测 -> 复习 -> 再测
- 每个模块都为下个模块提供输入，形成稳定闭环
- 这是现代语言学习产品里最核心的增长飞轮之一

### 产品结构设计建议（生产级版本）

#### 内容层
- **剧集素材库**：台词、时间轴、角色、场景、主题标签、难度标签
- **词汇层**：词义、词性、例句、搭配、近义表达、常见误区
- **语法层**：句式模板、替换练习、迁移表达
- **文化层**：语气、隐喻、美国职场文化、金融/法律语境

#### 学习引擎层
- **学习路径引擎**：根据水平和目标分配日计划
- **SRS 引擎**：复习调度
- **测评引擎**：自动组题、评分、错题归档
- **对话引擎**：苏格拉底式问题流 + AI 导师点评

#### 数据层
- **用户画像**：等级、薄弱点、掌握词汇、常错题型
- **行为数据**：学习时长、复习频次、正确率、连续学习天数
- **成长数据**：XP、等级、掌握曲线、阶段性报告

### 建议的下一步迭代

- **音频与跟读**：接入台词音频、TTS、跟读评分
- **听写模式**：隐藏文本，仅播放音频做听辨
- **AI 批改**：开放题和翻译题用 LLM 做更细粒度反馈
- **错题本**：将错误按词汇、语法、语境分类
- **学习推荐**：根据薄弱点动态推荐台词与题目
- **账户系统**：跨设备同步学习数据
- **内容后台**：支持编辑台词、批量导入剧集与字幕

### 商业化与增长建议

- **基础免费**：每日限量学习 + 基础测验
- **会员订阅**：完整剧集库、AI 导师、发音评分、个性化路径
- **学习社群**：打卡、排行榜、班级模式
- **B2B 场景**：高阶商务英语、金融英语、法务英语训练营

### 合规与版权建议

当前仓库中的台词数据是 **教学原型演示用样例数据**。如果产品要正式上线，建议：

- 使用自有授权内容或获得版权许可的字幕/台词资源
- 音频、视频片段必须单独评估版权授权范围
- 为素材管理建立可追溯的内容来源和授权台账

### 技术栈

- **前端**：React 19 + Vite
- **AI 驱动**：Google Gemini 1.5 Pro/Flash API
- **移动端**：Capacitor 8 (iOS/Android 支持)
- **路由**：`react-router-dom`（Hash Router）
- **存储**：浏览器 `localStorage`
- **样式**：原生 CSS 主题系统
